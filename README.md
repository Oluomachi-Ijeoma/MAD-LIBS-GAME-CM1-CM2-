Creating a Mad Libs game for CM1/CM2 students with little to no English background.

Purpose:
- To make learning English fun and interactive.
- To practice vocabulary and simple sentence structures from topics we have learnt, such as:
    - Daily routine (la routine)
    - Places (les endroits)
    - Family members (la famille)
    - Colours, numbers, days, months, and simple verbs

How it works:
- The program asks the student to input words (activities, family members, places, etc.).
- These words are inserted into a short story template using Python f-strings.
- After filling in the blanks, the full story is displayed for students to read and enjoy.

Learning objective:
- Reinforce vocabulary and sentence formation
- Encourage interaction with the language in a playful way
- Help students see how words can be used in context
